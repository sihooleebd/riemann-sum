# riemann-sum

사다리꼴 리만합과 직사각형 리만합의 오차에 대한 고찰 탐구를 통해
실제로 구현된 프로그램입니다.

여러 등분에 따른 사다리꼴, 왼쪽, 오른쪽 리만합을 구해주고 그래프로도 보여줍니다.

y=x 그래프의 0~1 범위에서 등분에 따른 각종 리만합
